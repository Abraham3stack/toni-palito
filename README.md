# Tony Palito – Football Player Portfolio

A professional football portfolio website showcasing Tony Palito, a creative and technically gifted central midfielder known for his vision, composure, and goal contributions.

## Features
- Player profile and bio
- Highlight videos
- Image gallery
- Contact section for trials and opportunities

## Tech Stack
- React / Vite
- Tailwind CSS

## Usage
1. Clone the repository
2. Install dependencies: `npm install`
3. Run locally: `npm run dev`

## Note
This project is built as a reusable template for football player portfolios. Content (name, images, videos, bio) can be easily replaced to create new player profiles.
